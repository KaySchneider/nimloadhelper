nimloadhelper
=============

nodejs image loader


loads all images from the file "loadImage"
every url is in an extra line.

Usage:
put your urls of the images into the "loadImage" file, seperated by
an "\n" new line.


on command-line:
navigate to the folder where the loader.js is.

type:
node loader.js
